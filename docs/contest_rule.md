## 2. 미션 시나리오 전체 흐름

```
① 자동 이륙 (C2)
      │
      ▼
② 바닥 구획선을 따라 2m 고도로 이동
   └ GPS 사용 금지
   └ Line Tracing 기반 이동
      │
      ▼
③ 경로점 4개 탐색
   └ 구획선 교차지점 무작위 배치
   └ 하향 카메라로 ArUco ID 식별
   └ 첫 인식 시 해당 마커 위에서 3초 호버링
   └ 각 경로점 위치 저장
      │
      ▼
④ 탐색 완료 후 ArUco 번호 오름차순으로 재방문
   └ 예시: ID 1 → 2 → 3 → 4 순서
   └ 구획선 추종 미검증 (직선 이동 가능)
   └ 재방문할 때마다 해당 마커 위에서 3초 호버링
   └ 각 경로점 통과 위치 정확도 평가
      │
      ▼
⑤ 시작점 (버티포트) 복귀
      │
      ▼
⑥ 자동 착륙 (C2)
```

> **C2 = Command and Control**: 자동 이륙/착륙은 지상 C2 명령으로 시작

## 3. 단계별 상세 임무 설명

### 3.1 ① 자동 수직 이륙

- 자동이륙 제어 (C2 명령으로 개시)
- 수직 이륙
- 구체적인 이륙 위치 = 버티포트 중앙

---

### 3.2 ② 바닥 구획선 정보 및 이동

**구획선 특성:**
- 경연장 바닥면 색상과 대비되는 색 사용 (시인성 확보)
- 폭 10cm
- 격자 크기 3m × 3m

**이동 조건:**
- 바닥의 구획선을 따라 일정 고도로 이동
- **GPS 위치 정보 사용 금지** (명시적 규정)
- Line Tracing 기능 구현 필수
- 고도 유지 정도 및 경로 이탈에 따른 차등 점수

**채점 포인트:**
- 고도 유지 (30점): 고도 침하 정도 기준
- 구획선 이탈 (40점): 정성 평가

---

### 3.3 ③ ~ ④ 경로점 탐색

**경로점 조건:**
- 경로점 수: **4개**
- 위치: 구획선이 교차하는 지점 (Intersection)
- 경로점마다 독립된 ArUco 마커 1개씩 배치
- ArUco 마커 크기: 50cm × 50cm
- **순차적 ID 부여** (예: 1, 2, 3, 4)
- 배치는 **무작위(랜덤)** — 사전에 알 수 없음
- 첫 인식 시 해당 마커 위에서 3초간 호버링 후 위치를 확정 저장

**경로점 식별 방법:**
- 하향 카메라 사용
- ArUco Marker ID 식별
- 위치 저장 (격자 좌표 또는 로컬 좌표계)

**채점 포인트:**
- 경로점 위치 추정 정확도 (30점 × 4개 = 120점)
- 경로점 식별 (ArUco ID 정확도) (10점 × 4개 = 40점)

---

### 3.4 ⑥ 구조 경로 — ArUco 번호 순서 재방문

**방문 순서 결정 방식:**
- ArUco Marker 식별번호 **역순 (내림차순)**
- 예시: ID 4 → 3 → 2 → 1

> 확정: 도식의 "⑥: 구조 경로(역순)" 표기가 올바름. 현재 mission_core 구현(역순)과 일치.

**이동 방식:**
- 구획선 추종 기능 **미검증** (규정 명시)
- 즉, 구조 경로에서는 격자선을 따르지 않고 **직선 이동 가능**
- 재방문할 때마다 해당 마커 위에서 3초간 호버링
- 각 경로점 통과 시 위치 정확도 및 순서 정확도로 채점

**채점 포인트:**
- 방문 순서 정확도 (40점)
- 경로점 통과 위치 정확도 (20점 × 4개 = 80점)

---

### 3.5 ⑦ 자동 착륙

- 자동착륙 제어 (C2 명령)
- 버티포트(시작점)로 복귀 후 착륙
- 착륙 위치 정확도에 대한 별도 채점 항목 없음 (규정집 기준)

### 4.2 항목별 채점 상세

#### (1) 경로점 위치 추정 — 120점 (30점 × 4개)

탐색 단계에서 드론이 마커를 인식하고 저장한 위치와 실제 위치 간의 오차를 측정.

| 등급 | X 오차 | Y 오차 | 점수 |
|------|--------|--------|------|
| 상 | < 0.5m | < 0.5m | 30점 |
| 중 | < 2m | < 2m | 18점 |
| 하 | < 4m | < 4m | 9점 |

**조건:**
- X, Y 오차 기준을 **동시에 만족**해야 해당 점수 부여
- **한쪽 기준만 만족** 시 해당 등급 점수의 **50% 부분 점수**

예시:
```
X 오차 0.3m (상 기준), Y 오차 1.5m (중 기준)
→ 동시 만족 불가 → 더 낮은 기준(중) 기준 50% = 9점
```

---

#### (2) 경로점 식별 — 40점 (10점 × 4개)

ArUco Marker ID를 정확하게 식별했는지 여부.

| 항목 | 내용 |
|------|------|
| 배점 | 10점 × 4개 = 40점 |
| 기준 | ArUco Marker ID 식별 정확도 |
| 방식 | 정답(O) / 오답(X) 이진 채점 추정 (세부 기준 미명시) |

---

#### (3) Line Tracing 성능 — 70점

탐색 구간 이동 중 구획선 추종 품질 평가.

**고도 유지 (30점):**

| 등급 | 고도 침하 정도 | 점수 |
|------|-------------|------|
| 상 | < 0.1m | 30점 |
| 중 | < 0.2m | 20점 |
| 하 | < 1m | 10점 |

> "고도 침하"란 규정 고도(2m)에서 아래로 벗어나는 정도를 의미.  
> 상향 이탈에 대한 기준은 별도 명시 없음.

**구획선 이탈 정도 (40점):**
- **정성 평가** (수치 기준 미제시)
- 이동 중 구획선에서 벗어나는 정도를 심판이 관찰하여 판단
- 이탈 횟수, 이탈 거리, 복귀 능력 등이 종합적으로 반영될 것으로 추정

---

#### (4) 구조 경로 추종 성능 — 120점

탐색 완료 후 재방문 구간 채점.

**방문 순서 정확도 (40점):**
- ArUco ID 오름차순 순서를 정확히 따랐는지 채점

**경로점 통과 위치 정확도 (80점, 20점 × 4개):**

| 등급 | 수평 오차 | 수직 오차 | 점수 |
|------|---------|---------|------|
| 상 | < 0.5m | < 0.5m | 20점 |
| 중 | < 2m | < 2m | 12점 |

**조건:**
- 수평/수직 기준을 **동시에 만족**해야 해당 점수

> "수직 오차"가 포함된 것은 구조 경로에서 고도 정확도도 함께 채점됨을 의미.

## 5. 위치 오차 허용 기준 상세

### 5.1 탐색 단계 — 마커 위치 저장 정확도

```
목표: 상 등급 (X < 0.5m AND Y < 0.5m)
최소 득점: 하 등급 (X < 4m AND Y < 4m) → 9점
부분 점수: 한 축만 만족 시 50%

개발 목표 기준:
  Grid Localizer 위치 추정 오차 < 0.5m (상 등급)
  Marker Confidence Filter 확정 시 위치 오차 < 0.5m
```

### 5.2 구조 경로 단계 — 경로점 통과 정확도

```
목표: 상 등급 (수평 < 0.5m AND 수직 < 0.5m)
최소 득점: 하 등급 (수평 < 4m AND 수직 < 4m) → 6점

주의: 수직 오차 포함
  → 고도 제어 정확도가 구조 경로 점수에 직접 영향
  → 2m 고도에서 ±0.5m 이내 유지 필요 (상 등급 기준)
```

### 5.3 고도 유지 기준

```
규정 고도: 2m
상 등급: 고도 침하 < 0.1m → 1.9m 이상 유지
중 등급: 고도 침하 < 0.2m → 1.8m 이상 유지
하 등급: 고도 침하 < 1m   → 1.0m 이상 유지

개발 제어 목표: 2m ± 0.1m (상 등급)
```

## 6. 개발 관점 핵심 제약 정리

### 6.1 GPS 사용 금지 (명시적 규정)

```
규정 원문: "드론은 GPS 기능을 활용하지 않고 바닥의 구획선을 따라서 2미터 고도로 비행"
```

- GPS 기반 위치 추정 불가
- Line Tracing + IMU + 하향 카메라 기반 항법 필수
- Grid-Based Localization 구현의 규정 근거

### 6.2 Line Tracing 필수 구현

```
규정 원문: "GPS 위치 정보 사용하지 않는 Line Tracing 기능 구현"
```

- 탐색 단계에서 구획선을 따라 이동하는 기능 필수
- 구획선 이탈 여부가 정성 채점 대상 (40점)
- 고도 유지 정도도 채점 (30점)

### 6.3 구조 경로 — 구획선 추종 미검증

```
규정 원문: "최적 구조 경로(역순)에 따라 경로점 이동 (구획선 추종 기능 미검증)"
```

- 구조 재방문 단계에서는 격자선을 따를 의무 없음
- **직선 이동 허용** → Sprint Planner에서 격자 그래프 제약 해제 가능
- 단, 위치 정확도와 고도 정확도는 여전히 채점

### 6.4 하향 카메라 필수

```
규정 원문: "하향 카메라를 사용하여 경로점의 ArUcoMarker 식별 및 위치저장"
```

- ArUco 식별에 하향 카메라 명시
- 마커 크기 50×50cm → 2m 고도에서 충분한 픽셀 크기 확보 가능

### 6.5 자동 이착륙 필수

- 이륙, 착륙 모두 C2 명령으로 수행하는 자동 제어 필수
- 수동 개입 없이 전 구간 자율 비행

### 6.6 방문 순서

```
확정: 역순 (내림차순) — 4 → 3 → 2 → 1
도식의 "⑥: 구조 경로(역순)" 표기가 올바름
```

- **역순 방문** (4→3→2→1) — 확정
- mission_core의 현재 구현과 일치, 변경 불필요

## 8. 점수 최적화 전략

### 8.1 점수 우선순위 분석

```
고정 점수 (노력 대비 확보 용이):
  경로점 식별 (40점): ArUco 인식만 정확하면 확보
  구조 순서 정확도 (40점): ID 저장 + 정렬 로직으로 확보

변동 점수 (정확도에 따라 상/중/하):
  경로점 위치 추정 (120점): Grid Localizer 정확도에 직결
  구조 통과 정확도 (80점): 저장 위치 재방문 정확도
  고도 유지 (30점): 고도 제어 튜닝

정성 점수 (구현 품질에 따라):
  구획선 이탈 (40점): Line Tracing 품질

상대 점수 (경쟁 팀 수준에 따라):
  임무 수행 시간 (100점): 빠를수록 유리

가산점 (자체 개발 어필):
  자체 개발 (50점): 문서화 및 발표 준비
```
